# Shape Links

You can embed links to specific shapes within text, allowing you to quickly navigate to them.

## Embedding a Shape Link

1. Select the target shapes and copy them using `Ctrl + c` (Windows) or `Command + c` (Mac).

   ![](/assets/shape-link-1.png)

2. Select the desired text and paste the link using `Ctrl + v` (Windows) or `Command + v` (Mac).

   ![](/assets/shape-link-2.png)

   ![](/assets/shape-link-3.png)

## Thumbnails

When the target shapes are in the same sheet, hovering over the link will display their thumbnails:

   ![](/assets/shape-link-4.png)
