# シェイプリンク

テキスト内に特定のシェイプへのリンクを埋め込むことで、素早く移動することができます。

## シェイプリンクの埋め込み

1. 対象のシェイプを選択し、`Ctrl + c`（Windows）または `Command + c`（Mac）でコピーします。

   ![](/assets/shape-link-1.png)

2. リンクを貼り付けたいテキストを選択し、`Ctrl + v`（Windows）または `Command + v`（Mac）でリンクを貼り付けます。

   ![](/assets/shape-link-2.png)

   ![](/assets/shape-link-3.png)

## サムネイル

対象のシェイプが同じシート内にある場合、リンクにカーソルを合わせるとサムネイルが表示されます：

   ![](/assets/shape-link-4.png)
